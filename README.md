# 智能苗圃监测与番茄病虫害AI预警系统

## 项目介绍
本项目是本科毕业设计，设计并实现了一套**基于ESP32-S3的物联网智能苗圃多参数监测系统**，结合深度学习技术，实现了番茄叶片病虫害的AI自动识别与预警。

系统主要功能包括：
1.  **环境数据采集**：通过ESP32-S3实时采集环境温度、环境湿度、土壤湿度数据
2.  **AI高清拍照**：双模式摄像头，准实时画面监控+AI级高清拍照
3.  **病虫害AI识别**：基于PlantVillage数据集训练的深度学习模型，支持10种番茄常见病虫害的识别
4.  **Web可视化界面**：ESP32本地Web页面展示实时数据与画面，Flask服务端展示AI识别结果与防治建议

## 硬件清单
- 主控板：ESP32-S3 CAM（带摄像头模块）
- 温湿度传感器：DHT11
- 土壤湿度传感器：电阻式土壤湿度传感器
- 其他：杜邦线、面包板、Type-C数据线

## 代码说明
 `github.py` | ESP32-S3硬件端主程序，负责传感器数据采集、摄像头控制、本地Web服务器搭建 
 `nursery_leaf_disease_ai_inference_demo.ipynb` | 番茄叶片病虫害AI识别推理代码，基于TensorFlow/Keras，支持单张图片的病害识别与结果可视化
 `simple_ai.ipynb`和`app.ipynb` | Flask Web服务端主程序，负责加载AI模型、提供病害识别网页界面、展示识别结果与专业防治建议

## 使用说明
### 1. ESP32硬件端部署
1.  安装Thonny IDE，配置ESP32-S3的MicroPython环境
2.  修改代码中的WiFi配置（`WIFI_SSID`和`WIFI_PASSWORD`）为你自己的2.4G WiFi信息
3.  将代码烧录到ESP32-S3，按硬件接线说明连接传感器与摄像头
4.  上电启动，通过串口监视器输出的IP地址访问本地Web页面

### 2. AI识别服务端部署
1.  安装Python 3.8+环境
2.  安装依赖包：`pip install tensorflow opencv-python flask numpy matplotlib pillow`
3.  准备训练好的`.h5`模型文件（网盘链接），修改代码中的模型路径
4.  运行Jupyter Notebook代码，即可执行AI识别推理
